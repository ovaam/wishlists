# Настройка Firebase для синхронизации данных

## Проблема
Сейчас данные сохраняются только в браузере каждого пользователя (localStorage), поэтому желания не видны другим.

## Решение: Firebase Realtime Database

Firebase - бесплатный сервис от Google для синхронизации данных в реальном времени.

### Шаг 1: Создать проект Firebase

1. Зайдите на https://console.firebase.google.com
2. Нажмите "Add project" (Добавить проект)
3. Название: `wishlist-app` (или любое другое)
4. Отключите Google Analytics (не обязательно)
5. Нажмите "Create project"

### Шаг 2: Создать Realtime Database

1. В меню слева выберите "Realtime Database"
2. Нажмите "Create Database"
3. Выберите регион (ближайший к вам)
4. Выберите режим: **"Start in test mode"** (для начала)
5. Нажмите "Enable"

### Шаг 3: Получить конфигурацию

1. В настройках проекта (шестеренка) → Project settings
2. Прокрутите вниз до "Your apps"
3. Нажмите на иконку Web (</>)
4. Зарегистрируйте приложение (название: `wishlist`)
5. Скопируйте конфигурацию (она будет добавлена в код)

### Шаг 4: Настроить правила безопасности

1. В Realtime Database → Rules
2. Замените правила на:
```json
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
```
⚠️ **Внимание:** Это открывает доступ всем. Для продакшена нужно добавить аутентификацию.
